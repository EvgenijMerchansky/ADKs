# JsonPlaceholderADK (Agent Development Kit)
Simple ADK application, which works with https://jsonplaceholder.typicode.com/

# Local development:

# - create venv
`py -m venv .venv`

# - activate env
`source .venv/bin/activate` **(macOS/Linux)**

`.venv\Scripts\activate.bat` **(CMD)**

`.venv\Scripts\Activate.ps1` **(PowerShell)**

# - packages
`python -m pip install -r requirements.txt` **(Bash)**

`py -m pip install -r requirements.txt` **(PowerShell)**

# - before run
`cd ..`

# - run
`adk web`
